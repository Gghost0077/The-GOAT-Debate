# The GOAT Debate

A React + TypeScript web app exploring the Messi vs Ronaldo debate — comparing the careers and achievements of two of football's greatest players, with a planned voting feature for users to weigh in.

## Features

- Responsive navigation bar with a condensed mobile view
- Image carousel showcasing both players
- Login page (in progress)
- Built with Bootstrap for styling alongside custom components

## Planned Features

- Dedicated info pages for each player's career and achievements
- User login/signup system
- Voting system for users to cast their vote on the GOAT debate

## Tech Stack

- React 18 with TypeScript
- Vite for development and build tooling
- Bootstrap for styling
- ESLint for code quality

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Run the development server:
   ```
   npm run dev
   ```

## Status

This project is a work in progress — the homepage, navigation, and carousel are in place, with the login/signup and voting systems planned next.
